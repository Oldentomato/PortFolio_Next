## 변수 실수형  
값이 너무 큰 변수는 0이 많아 숫자가 너무 길고 값을 파악하기도 헷갈리며,  
두 값을 비교하는 것도 어렵다. 이럴 때 부동소수점 방식을 사용한다.  
    (가수 E 지수)
가수는 값의 모양을 나타내고 지수는 값의 크기를 나타낸다.  
(예) 9조 4600억 -> 9.46e12 (9.46 * 10의 12승이다.)  
## 변수 복소수형  
복소수는 제곱해서 음수가 되는 가상의 숫자이다. 실제로 존재하지 않는 허수여서 실생활에서 쓸일은 없지만 과학분야에서는 중간 과정계산을 위해 종종 사용된다.  
    (실수부 + 허수부j)
(예) a = 1 + 2j
    b = 3 + 4j
    print(a + b)
    >> (4 + 6j)
## 문자열 관련  
문자열은 큰 따옴표 안에서 이루어져야 하는데 만일 내부에 큰 따옴표를 쓰고 싶다면 작은 따옴표로 문자열 이루고 쓰면 된다.  
반대로 작은 따옴표는 큰 따옴표 내에 있어도 상관없다.  
(예) "Let's go"  
다음과 같은 확장열을 이용하여 표현할 수도 있다.  
- \n : 개행  
- \t : 탭  
- \" : 큰따옴표  
- \' : 작음 따옴표  
- \\ " \문자  
문자열을 작성할 때 긴 문자열을 개행하여 쓰고 싶을 때는 따옴표를 세개 쓰면 된다. (큰, 작은 상관없음)  
(예) """ 엄청 긴 문자열 ...
        ...
        ... """
이걸 쓰기 싫다면 문자열 끝마다 \를 써주어도 된다.  

## 컬랙션  
정수, 실수, 문자열 등의 단순 변수는 단 하나의 값만을 저장하는데 비해 컬랙션은 여러 개의 값을 모아서 저장한다.  

## 가변 인수  
인수 이름 앞에 * 기호를 붙이면 여러 개의 인수가 올 수 있다.  
```python
def intsum(*ints):
    sum = 0
    for num in ints:
        sum += num
    return sum
print(intsum(1,2,3))
print(intsum(5,7,9,11,13))
```
인수를 받아올 때는 튜플로 받아온다.  
단, 가변인수는 이후의 인수를 다 포함하기 때문에 인수목록의 마지막에 와야 한다.  
```python
intsum(s, *ints) #가능
intsum(*ints, s) #불가능
intsum(*ints, *nums) #불가능
```

## 키워드 가변인수
키워드 가변인수를 가변 개수 전달할 때는 인수목록에 **기호를 붙인다.  
호출원에서 여러 개의 키워드 인수를 전달하면 인수의 이름과 값의 쌍을 사전으로 만들어 전달한다.
