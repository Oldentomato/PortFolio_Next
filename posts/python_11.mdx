## 문자열 첨자  
개별문자를 읽을 때는 []괄호와 문자의 위치인 첨자를 적는다. 첨자는 앞에서 셀 수도 있고 뒤에서 셀 수도 있다.  
```
앞에서 셀 때 -> 0  1  2  3  4  5   
             p  y  t  h  o  n  
            -6 -5 -4 -3 -2 -1 <- 뒤에서 셀 때  
```
  
## 문자열 슬라이스  
[] 괄호에 첨자를 하나만 적으면 해당 위치의 문자를 읽지만 범위를 지정하면 부분 문자열을 추출한다.  
[] 괄호 안에 시작, 끝, 증가값을 지정하는데 range 함수와 구조가 같다.  
[begin : end : step]  
  
## 문자열 검색  
find()와 rfind()가 있는데, rfind는 뒤에서부터 찾는 함수이다.  

## 문자열 조사  
특정문자가 어디에 있는지 관심없고 단순히 있는지 없는지 알고 싶을 때는 in 구문을 사용한다.  
```python
s = "python programming"
print('a' in s)
# 실행결과 : true
```
startswith() 메서드는 특정 문자열로 시작되는지 조사한다.  
반대로 endwith() 메서드는 특정 문자열로 끝나는지 뒤부분만 비교한다.  
```python
name = "김한결"
if name.startswith("김"):
    print("김가입니다.")
if name.startswith("한"):
    print("한가입니다.")

file = "girl.jpg"
if file.endswith(".jpg"):
    print("그림 파일입니다.")
```
  
## 문자열 공백  
의미없는 공백을 제거할 때는 다음 메서드를 사용한다.  
```python
s = "   angel   "
s.lstrip() #왼쪽 공백만
s.rstrip() #오른쪽 공백만
s.strip() #양쪽 공백
```
  
## 문자열 분할  
splitlines() 메서드는 개행 문자나 파일 구분자, 그룹 구분자 등을 기준으로 문자열을 잘라 리스트로 만든다.  
join() 메서드는 split()메서드와 반대로 문자열의 각 문자 사이에 다른 문자열을 끼워 넣는다.  
split()과 join()을 응용하면 다음과 같은 코드를 작성할 수 있다.  
```python
s2 = "서울->대전->대구->부산"
city = s2.split("->")
print(" 찍고 "join(city))
```
실행결과: 서울 찍고 대전 찍고 대구 찍고 부산
