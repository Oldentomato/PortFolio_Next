```console
zsh: command not found: python
```
python을 설치했는데도 위와 같은 에러가 뜬다면 환경변수에 python이 설치된 경로가 설정되지 않아서 설치된 python을 알지못하여 나는 에러이다.  
해결방법은 환경변수를 지정해주면 된다.  

```console
#python이 설치된 위치를 찾습니다.
which python3

#환경변수 파일에 경로 지정
vi ~/.bash_profile

#그 후 수정사항을 적용하기 위해
source ~/.bash_profile
```
적용 후 파이썬을 실행하면 잘 실행된다.

```console
sudo pip install something[any] 명령이 no matches found
```
위의 에러는 pip로 패키지를 설치할 때 나오는 에러이다.  
원인은 zsh이 square brachet을 globbing, pattern matching용으로 사용하기 때문이다.  
패키지명을 따옴표로 감싸주면 해결된다.  
```console
sudo pip install 'ansible-container[docker,k8s]'
```
