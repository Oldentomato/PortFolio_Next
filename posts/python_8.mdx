mlops 공부중에 이러한 문법을 발견했다.

```python
if __name__ == "__main__":
    function_name()
```

__name__이라는 변수는 원래 파이썬에 내장되어 있는 변수라고 한다.  
실행시키는 해당 파일안에서 함수를 실행시키면 __name__변수에 담기는 것은 __main__으로 지정되어있다.  
즉, 위의 코드는 해당 파일 자체를 실행하는 경우 함수이름을 실행하는 것이고,
다른 파일에서 실행한다면 함수이름은 실행되지 않는다.
